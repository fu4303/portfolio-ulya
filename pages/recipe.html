Recipe Generator created with
React, Bootstrap, Firebase Auth, and TheMealDB API.

Live GitHub

Random recipe generator is a simple app created with React, TheMealDb API, and Firebase authentication. I designed it with Bootstrap.

This project was a result of my very first steps in learning React. I was looking for an API to practice data fetching. When I found TheMealDb I was excited to build something that I can use in my daily life for cooking inspiration.

Later on, I added category filter and Firebase authentication to allow users to add selected recipes to favorites.

Features:

  • - Get a random recipe for the selected category
  • - Sign up, log in, reset forgotten password
  • - Add a recipe to favorites
  • - Delete a recipe from favorites
